# Lesson 07 — Verify Three Truths

[← Previous: 06](06-execute-smallest-step.md) · [Journey](../JOURNEY.md) · [Next: 08 →](08-review-and-explain.md)

Time: 10–15 minutes

## Learn

A focused test proves one behavior. It does not prove the repository is healthy or that the final claim is accurate.

## Do

Verify:

1. **Code truth:** inspect the exact diff.
2. **Data truth:** run a probe for blank and valid titles.
3. **Decision truth:** compare results to the four-line contract.
4. **Regression:** run the full test suite.

## Gate

- **Explain:** Give an example of a passing test that would not prove the final claim.
- **Execute:** Run all four checks.
- **Verify:** Report exact commands and outcomes, plus anything not verified.
