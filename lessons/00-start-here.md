# Lesson 00 — Start Here

**First lesson** · [Journey](../JOURNEY.md) · [Next: 01 →](01-name-the-loop.md)

Time: 10–15 minutes

## Learn

This boot camp measures learning by what you can explain, execute, and verify—not by time spent or pages read.

## Do

From the repository root:

```bash
mkdir -p learner
cp templates/learner-progress.md learner/progress.md
python3 -m unittest discover -s tests -v
```

Record the test result in `learner/progress.md`.

## Think

Without rereading the README, answer:

1. Why is `learner/` ignored by Git?
2. Why does this repository begin with a toy project rather than your real production code?
3. What are the three kinds of mastery evidence?

## Gate

- **Explain:** Answer the three questions.
- **Execute:** Create the private progress file.
- **Verify:** Show the baseline test command and result.

When ready, ask your instructor to review Lesson 00 evidence.
