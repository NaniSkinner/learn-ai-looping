# Lesson 10 — Foundation Capstone

[← Previous: 09](09-preserve-the-lesson.md) · [Journey](../JOURNEY.md) · **Foundation capstone**

Time: 20–30 minutes; pausing is allowed

## Learn

Transfer means operating the loop on a new task with less scaffolding.

## Challenge

Add a `completion_rate` field to `summarize_tasks`:

- `0.0` when there are no tasks,
- otherwise `completed / total`,
- preserve the existing count invariant.

## Do

Run the entire loop without copying the earlier answers:

Frame → Orient → Predict → Plan → Execute → Verify → Explain → Preserve

Your instructor may organize and question, but should not supply the implementation before your first attempt.

## Graduation gate

- A four-line contract exists before planning.
- The pre-agent prediction is recorded.
- The agent plan was compared, not automatically accepted.
- A new test failed before the fix.
- Focused and full verification passed afterward.
- The learner can explain the change without the transcript.
- One durable lesson was placed deliberately.
- The learner can name at least one situation where this loop should be shortened.
