# Lesson 04 — Predict Before Prompting

[← Previous: 03](03-orient-to-truth.md) · [Journey](../JOURNEY.md) · [Next: 05 →](05-plan-and-compare.md)

Time: 10–15 minutes

## Learn

Prediction forces your brain to build a model before the agent gives you one. The goal is not to be right; it is to make changes in your understanding visible.

## Do

Before asking for a plan, predict:

- files likely to change,
- smallest implementation change,
- test that should fail before the fix,
- behavior that must remain unchanged,
- one tempting but unnecessary change.

Assign confidence—low, medium, or high—to each prediction.

## Gate

- **Explain:** Why is a wrong written prediction useful?
- **Execute:** Record all five predictions.
- **Verify:** Your prediction is timestamped before the agent plan.
