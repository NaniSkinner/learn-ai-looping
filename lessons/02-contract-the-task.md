# Lesson 02 — Contract the Task

[← Previous: 01](01-name-the-loop.md) · [Journey](../JOURNEY.md) · [Next: 03 →](03-orient-to-truth.md)

Time: 10–15 minutes

## Learn

Before the agent plans, write:

- **Goal:** the outcome to create.
- **Context:** the source material that matters.
- **Constraints:** boundaries, risks, and authority.
- **Done when:** observable completion evidence.

## Challenge

The practice project currently accepts a blank or whitespace-only task title. The desired behavior is to reject it with `ValueError` while preserving all existing behavior.

## Do

Write a four-line contract for this challenge. Do not ask the agent to solve it yet.

## Gate

- **Explain:** Why is “fix blank tasks” an incomplete request?
- **Execute:** Produce the four-line contract.
- **Verify:** Every done-when statement must be observable by a command, diff, or test.
