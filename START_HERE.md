# Start Here

This is the only page you need to begin.

## 1. Set up your private learner space

From the repository root:

```bash
mkdir -p learner
cp templates/learner-progress.md learner/progress.md
python3 -m unittest discover -s tests -v
```

The `learner/` directory is ignored by Git. Your answers, questions, mistakes, and private examples stay out of the public repository.

## 2. Begin the first lesson

Open **[Lesson 00 — Start Here](lessons/00-start-here.md)**.

Lesson 00 asks you to record the baseline test result and answer three short questions. Give that evidence to your AI or human instructor.

## 3. Resume at any time

When returning later:

1. Open `learner/progress.md`.
2. Read only the current lesson and exact next action.
3. Try to retrieve the prior idea before reading your notes.
4. Use [JOURNEY.md](JOURNEY.md) only when you need the complete map.

## Time available right now

- **5 minutes:** retrieve one concept, answer one question, or make one prediction.
- **10–15 minutes:** complete one lesson gate.
- **30–60 minutes:** complete several lessons or one entire loop.

Time does not unlock the next lesson. Evidence does.
